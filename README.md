# VPN-SITE-TO-SITE
IPsec Site-to-Site VPN implementation using FortiGate firewalls on PNETLab. This project demonstrates a full security lifecycle: enrollment-based addressing (2024-2372), Phase 1 (IKE), Phase 2 (IPsec), static routing, and firewall policies. Validated with end-to-end connectivity tests via trace-route and ICMP ping.
